# Project 1 — AI-Assisted Knowledge Base Article

## Project Overview

This project demonstrates how a common SaaS support issue can be converted into a structured knowledge base article.

The goal is to reduce repetitive support requests by providing clear self-service documentation for users.

## Scenario

Users frequently contact support because they cannot reset their password.
Common issues include expired reset links, missing emails, or attempts to reuse old reset links.

## Documentation Approach

This project demonstrates a typical documentation workflow:

1. Identify a recurring support problem.
2. Analyze the user friction points.
3. Create structured documentation that enables users to resolve the issue independently.

## Files in This Project

**original_support_problem.md**
Describes the support scenario that triggered the documentation.

**knowledge_base_article.md**
The help center article written to address the issue.

**reflection.md**
Explains the documentation decisions, user experience considerations, and how AI-assisted drafting was used in the workflow.
